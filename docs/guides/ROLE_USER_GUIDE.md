# User Guide by Role and Use Case

## Admin / CEO / Leadership
Use cases:
- Cross-department audits
- Risk reviews
- Executive summaries

Behavior:
- Broad access across departments
- Can view admin endpoints and audit logs

## Finance / Finance Lead
Use cases:
- Quarterly reporting
- Revenue, margin, forecasting analysis

Behavior:
- Finance-focused content access
- Blocked from HR-restricted content

## HR / HR Lead
Use cases:
- Policy guidance
- Employee process and compliance lookups

Behavior:
- HR-focused content access
- Blocked from finance-restricted content

## Engineering / Tech Leads
Use cases:
- Architecture and platform guidance
- Dev standards and implementation lookup

Behavior:
- Engineering-focused content access

## General Employees
Use cases:
- Announcements
- General handbook and broad policy lookups

Behavior:
- Access to general/allowed content only
- Restricted departmental content is hidden

## Chat Best Practices
- Ask focused, department-specific questions
- Verify answer citations and snippets
- Treat low confidence responses as hints and refine question
